#### 一、初识nodejs

node.js是一个异步的事件驱动的JavaScript进行时

特性： 非阻塞i/o,  事件驱动

为性能而生： 多进程-c, 多线程-java， 异步IO - js, 协程 - TS

1. helloWorld.js

   ```
   console.log('hello,node.js!'); 
   console.log('run me use: node helloWorld!');
   ```

   > 每次修改js代码后,需要频繁的手动close掉,然后再重新启动,非常繁琐。安装nodemon可以监视文件改动，自动重启
   >
   > npm i -g nodemon
